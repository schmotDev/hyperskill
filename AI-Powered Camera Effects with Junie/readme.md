##  AI-Powered Camera Effects with Junie

This project was built with the AI assistant JetBrain Junie as a plugin to the JEtbrain WebStorm IDE.


### Summary

The web page generated is using the camera of your computer and propose few filters that can be applied direcvtly on the video.
If you save an image from the live video, the filter is also applied.
Some filters work better with Chrome.


### To test the project
* You can simply open the `index.html` file in your browser.
