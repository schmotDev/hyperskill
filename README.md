# Hyperskill

A small repository for learning and experimenting with programming exercises and projects.

## About

This repository contains practice exercises, example projects, and notes used for learning software development concepts. It's intended to be a personal workspace for experimentation and study.

## Features

- Collection of small practice projects
- Example code and notes
- Lightweight, educational-focused structure

## Getting started

Prerequisites:
- Git
- A programming language/runtime appropriate for the exercises (e.g. Python, Java, JavaScript)

Clone the repository:

```
git clone https://github.com/schmotDev/hyperskill.git
cd hyperskill
```

Review the folders and open the example projects or exercises. Each project or exercise should include its own README or usage notes where necessary.
