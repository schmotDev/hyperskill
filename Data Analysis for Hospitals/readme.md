## Data Analysis for Hospitals

A comprehensive data study using the pandas library.

<br>

### Summary

The script will upload datasets and correct errors in the csv files, deal with data omissions and incorrect data filling, find the main statistical characteristics, and create different graphs and plots to visualize the data.
<br>
- Load patient data from three hospital CSV files
- Standardize column names and merge all datasets into one
- Remove empty rows and clean incorrect or missing values
- Normalize categorical data (gender, tests, diagnoses)
- Analyze patient statistics across hospitals
- Visualize age distribution, diagnoses, and height by hospital
<br><br>

### To run the script
* Simply run the script

```bash
python analysis.py
```
